
# TrblTours

TrblTours is a Laravel 13 application for local tour discovery and guide management.

## Initial Development

The project started on April 6, 2026 with a full Laravel scaffold: auth, dashboards, profile management, database migrations, factories, Tailwind/Vite assets, and Pest/Dusk testing.

The first major product work followed on April 8, 2026 and focused on signup persistence, admin authentication, role-based access, and dashboard flows for different user types.

From there, development shifted toward guide-facing features such as tour management, booking-related views, profile pages, settings, and richer dashboard interactions.

## UI Direction

The app uses a warm travel-oriented style with sand, gold, and olive tones, rounded cards, soft shadows, and large scenic imagery.

Typography centers on Instrument Sans for body text, with Asimovian and Cal Sans used for branding and headline treatment.

## Stack

- Laravel 13
- Livewire 3
- Tailwind CSS
- Vite
- Pest
- Dusk
